# ContinualLearning2026
継続学習用のプログラムです．

# プログラムの全体像
学習・評価に使用するプログラムの全体像は以下の通りです．
各ディレクトリの詳細については，それぞれのディレクトリのREADMEを確認してください．
```
SSOCL/
├── buffer
├── configs           : 学習・評価の設定を記述する.yamlファイルの格納場所．
├── criterions
├── dataloaders
├── main.py
├── models
├── optimizers
├── trainers
└── utils.py          : その他のモジュールを実装するutilsファイル．
```
